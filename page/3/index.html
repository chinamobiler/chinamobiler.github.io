中国化工未来5年6大发展

发表于
1月 1 2015

从国家经贸委近日在成都召开的“全国石油和化工行业工作座谈会”上获悉，未来5年中国化学工业将针对目前存在的突出问题，重点做好6个方面的工作，力争实现产值年均增长7％；主要产品产量平均增长6％；高浓度化肥、低毒高效农药、离子膜烧碱、子午线轮胎在行业中的产量比重分别达到65％、55％、40％和50％的目标，使中国化学工业总体水平再上新台阶。　　据了解，中国化学工业经过50年的发展，已取得显著成果。但依然存在结构性矛盾突出，机制转换缓慢，科研基础薄弱，“三废”治理率低等问题。对此，国家经贸委提出中国化工应重点发展的6个方面。　　一是继续重点发展农用化学品。化肥行业重点是提高国产化肥的竞争能力，保证其在国内市场的主导地位。其中氮肥要抓好大、中型企业的节能、增产和提高产品质量的技术改造；磷复肥要大力增产，向规模化、基地化方向发展；钾肥是重点建设好青海钾肥基地和积极利用国外资源，缓解中国钾肥供应压力。农药工业重点是进行产品结构调整和企业组织结构调整，加快替代高毒有机磷的杀虫剂、除草剂和杀菌剂的发展，采取多种措施逐步使企业走向集约化经营。　　二是搞好国民经济重点产业的配套。重点发展有机硅、有机氟、聚碳酸酯、聚甲醛、聚氨酯等化工新材料；子午线轮胎、汽车及机械用新型橡胶制品及为国防军工配套的橡胶制品、建材用塑料制品和防水材料等；建筑、汽车、交通用新型高档涂料、电子工业用无机化工材料等。　　三是加快精细化工产品的发展，提高化工行业的精细化率。重点发展经表面处理的、高纯、晶型单一的、超细和纳米级无机盐产品；节能、环保型、耐久型及具有特殊功能的专用涂料；符合国际纺织品环保法规要求的分散染料、活性染料、酸性染料以及对国民经济有重要支撑作用的新领域精细化工产品。　　四是加强同石油化工的结合，优化原料和技术路线。以乙烯法PVC替代电石法PVC为突破口，在基础有机原料和氯丁橡胶主要产品生产中，全面以石化原料路线替代传统原料路线。　　另外两项发展重点是加强节能、环保与资源节约以及发挥西部优势，加快西部化工发展。 2002年2月25日

    国务院办公厅就全国认证认可工作发出通知要求全面加强和规范认证认可工作

    发表于
    1月 1 2015

    为适应社会主义市场经济发展的需要，促进我国认证认可工作健康、有序地发展，国务院办公厅日前发出了《关于加强认证认可工作的通知》。 《通知》阐述了加强认证认可工作的重要意义和指导思想，明确了我国认证认可工作当前的主要任务和工作重点，并就做好今后的认证认可工作提出了统一步调，形成合力的要求。 《通知》指出，我国认证认可工作的指导思想是按照“三个代表”重要思想的要求，坚持统一规划、强化监管、规范市场、提高效能和与国际接轨的原则，建立国家认证认可监督管理委员会（以下简称国家认监委）统一管理、监督和综合协调下，各有关方面共同实施的工作机制，建立全国统一的国家认可制度和强制性认证与自愿性认证相结合的认证制度，积极推进认证机构及相关机构的改革，逐步实现认证机构的社会化。 就我国认证认可工作当前的主要任务和工作重点，《通知》强调了四个方面：建立统一的国家认可制度；建立适应市场经济发展要求的认证制度；整顿和规范认证市场；建立健全的工作制度，完善法制。 《通知》进一步指出，认证认可工作涉及面广，专业性、技术性强，需要各有关部门的共同努力和密切协作。各有关部门要积极支持国家认监委的工作，在国家认监委统一规划与有效监管的基础上，各负其责，密切配合，共同搞好我国的认证认可事业。尽快建立认证认可工作部际联席会议制度，协调动作，形成合力。 2002年2月22日

    煤科总院上海分院开发环氧树脂设备

    发表于
    12月 31 2014

    　　由煤科总院上海分院研制的HAG型全液压环氧树脂自动压力凝胶成型机，已在全国10多个省市投入使用，年产值达到3000多万元。该设备市场占有率已达80％以上，居全国第一。　　环氧树脂类设备以前大都依赖进口，价格贵得惊人，按可比价格计算，开发生产环氧树脂设备每年可为国家节省3000多万美元。受国家环氧树脂APG项目组的委托，煤炭科学研究总院上海分院液压研究所以庞鸣崴为主的课题组，自筹资金，凭借一组引进设备的照片和样本，开始了环氧树脂设备的研制工作。经过7年多的努力，终于完成了施工图纸设计、样机制造，并投入工业性生产。1997年，HAG887型成型机通过国家机械部部级鉴定。2001年，HAG888－V型成型机通过上海市科委的鉴定。该类设备先后荣获上海市“四新”产品博览会金奖、上海市第五届科学技术博览会金奖、上海科学院技术进步一等奖。上海分院环氧树脂项目部近期正在着手开发硅橡胶成型设备、互感器、复合绝缘套管等新的项目，以加强技术储备，增强发展后劲。 2002年1月24日

    软包装的2002年

    发表于
    12月 31 2014

    　　到2002年，软包装在食品业的销售额将超过包装加工业销售额的一半以上，其余的为消费品工业市场。销售额分配在易腐食品和食品上。　　到2002年，袋装内食品以每年20%的速度成为增长最快的部分。使用的主要原料将包括清洁、防雾的封闭材料、高阻隔性薄膜以及广泛使用的收缩薄膜。到2002，在鲜肉包装中，将有近1/3的部分将使用袋式包装。　　当然，包装要求的市场特性、大小和复杂性将造成一段时期内包装材料的多样化。　　随着早餐及宴会用食品的畅销，其包装薄膜将以每年9%的速度成为增长最快的部分。茂合金属作为密封材料正逐渐代替离聚物或乙烯醋酸乙烯酯。尽管一些食品的销售由于其高脂肪以及早餐食用它的人数减少等原因而逐渐减少，但口味、低脂肪的品种仍有助于促进包装薄膜的需求增长。　　对于许多市场，特别是快餐和糖果市场而言，新颖的图案是必要的，但其更要求包装具有较高的防潮性能。这对包装薄膜市场也许具有较为积极的影响。市场还包括谷类食品、宠物食品及咖啡。　　工业品的销售量和销售额是最大的一个领域。保健类包装产品的增长速度最快。使用软包装材料的产品还包括肥皂、化妆品、一次性纸品、及其他零售品。　　总之，柔性包装已渗透到制造领域的每个部分。由硬包装转化而来的柔性包装将继续发展。但速度会慢下来，因为许多最易转化的工作已完成了。预计柔性包装总的发展速度要略快于总的国民产品的发展速度。商情快递2001年12月12日

    大连将出现塑料啤酒瓶

    发表于
    12月 31 2014

            一种可以回收利用具有较强的环保性的塑料啤酒瓶，即将在辽宁大连出现。    据了解，目前国外一些大的啤酒厂，都开始部分生产塑料瓶装啤酒，而且技术越来越先进的新型塑料瓶，不仅保持了啤酒自身的香气，还可以回收利用，具有较强的环保性。    据介绍，随着塑料啤酒瓶存在的阻隔性、啤酒风味和贮藏期限等问题的解决，用塑料瓶装啤酒很快会成为现实。中国新闻社 2001年12月12日

    中国实现塑料原料进口货源多元化

    发表于
    12月 31 2014

    过去中国进口塑料原料对日本和韩国产品的依赖程度高，如20世纪90年代中期前，自韩国的进口量占进口总量的50%。中国努力实现合成树脂进口货源多元化，自1998年韩国产品下降为占30.6%，今年下降为占23.4%，而聚乙烯、聚丙烯韩国产品下降为占14%。与此同时，我自台湾盛东南亚、中东国家的进口量逐年大幅增长，它们的产品在中国进口总量中上升为占33%以上。 (中国开门化工商务网) 2001年12月14日

    塑料建材的市场前景相当广阔

    发表于
    12月 31 2014

    据专家预测，“十五”期间，各种建筑塑料管和塑料门窗全国平均市场占有率将分别达到45％和20％，需要各种塑料管与门窗型材约100万吨，加上高分子防水材料、装饰装修材料、保温材料及其他建筑用塑料制品，总需求量约为400万吨。 　　塑料建材在建筑工程、市政工程、村镇建设以及工业建设中用途十分广泛。它不仅能大量代钢代木，替代传统建材，而且还具有节能节材、保护生态、改善居住环境、提高建筑功能与质量、施工便捷等优越性。塑料建材的节能效益十分突出：建筑塑料制品仅为钢材、铝材生产能耗的25％和12．5％，硬质PVC塑料生产能耗仅为铸铁管和钢管的30％至50％，塑料给水管比金属管降低输水能耗50％左右。 　　今后5年至15年内，我国建筑业将会有较大的发展。目前塑料建筑排水管已较普遍得到应用。今年塑料管与塑料门窗全国建筑市场平均占有率将分别达到30％和15％。 (中国开门化工商务网) 2001年12月14日
