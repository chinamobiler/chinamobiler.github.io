Siemens集成电路载体首次采用热塑性塑料

发表于 12月 31 2014

Siemens(西门子)Demantic公司制备的PSGA一体化集成电路外壳是第一次用热塑性塑料做集成电路载体，只比集成电路本身大20%，是用LCP（液晶聚合物）注塑而成。由于LCP的耐高温性、尺寸稳定性、良好的熔体流动性和介电性能使其适于这种高性能应用。美国Du Pont和Ticona公司都供应这种要求的LCP. 外壳底部直径0.4mm、高约0.4mm的接头用高精度注射成型与整个部件底板一次成型制备，每次最多可成型64个底板，每个重量不到1g。LCP部件尺寸稳定性和耐高温性确保接头在无铅焊料回流焊接时高达260℃温度时，仍十分精确地保持在原有位置。LCP与集成电板热膨胀系数相近，故十分可靠，LCP的柔性可补偿外壳和集成电路板尺寸变化带来的误差，温度反复变化时，不会由于导电元件机械应力而产生误动作。用LCP不仅成本低，而且生产步骤少。 2002年2月1日

    中国公布对内外产品统一认证法规

    发表于 12月 31 2014

    中国将从2002年5月1日起实施新的强制性产品认证制度。国家质量监督检验检疫总局和国家认证认可监督管理委员会今天正式公布了国家强制性产品统一认证的有关法规文件，从而终结了以往内外产品认证制度不一致的局面。 为履行入世承诺，中国政府合并了原国家质量技术监督局，组建国家质量监督检验检疫总局和国家认证认可监督管理委员会。由认监委统一负责国家强制性产品认证认可。 据悉，此次被列入目录的强制性认证产品有132种；使用统一的“ＣＣＩＢ”标志，逐步取代原来的“长城”和“ＣＣＩＢ”标志；收费将体现不以赢利为目的和国民待遇原则。 2002年2月1日

    《温州悬念》首发

    发表于 12月 31 2014

    温州晚报讯 中国社科院前不久首次披露的关于当代中国社会十大阶层的研究报告引起了各界的广泛关注。而全国第一本以阶层分析的视角，立足于极为丰富的现实生活，对一个个性化的区域进行全方位透视的力作《温州悬念》又与读者见面。昨天，浙江人民出版社在杭州举行了该书的首发式。 　　《温州悬念》的作者是两位常驻浙江的新华社记者，多年来追踪温州现象。他们以记者特有的职业眼光，在独创的理论构架平台上，发掘了鲜活的社会真实。该书史料性强，几乎触及了二十多年来温州改革的全部著名人物和著名事件。同时，还对1989年前后中南海三次派员下温州、1995年北京清理“浙江村”、股份合作制制度创新的台前幕后、温州富人的财富状况等作了全面、权威的揭秘。 　　（古月） 2002年1月31日

    长春应化所新型蓝光转化地膜可使水稻显著增产

    发表于 12月 31 2014

    中科院长春应用化学研究所研制成功新近研制成功的蓝光转化膜日前通过了中国科学院组织的专家验收。鉴定意见称，使用该地膜可使水稻最多增产达９％。经过在吉林、辽宁、江苏、浙江、湖南等气候、地理条件差别很大的地区进行试用，增产效果均十分明显。　　据课题组介绍，蓝光转化膜因为可以更多更好地利用太阳能促进农作物增产增收，而成为当前农膜研发的热点。中科院长春应化所从１９９７年开始对水稻育秧蓝光转化膜进行应用研究，成功研制出稀土有机转光剂，解决了技术难题。同时科研人员还制备出适用不同水稻品种和南北方不同日照时间的系列蓝光膜。 据了解，该科研成果目前已成功实现转化。在吉林省辽源市塑料四厂建立的生产基地已形成年产５００吨蓝光膜母料和５０００吨蓝光膜的能力。 2002年1月31日

    国外塑料机械工业未来发展的总体趋势

    发表于 12月 31 2014

    　　国外塑料机械工业未来的总体发展趋势，是朝着精密、高质、高效、节能、低噪音及可持续发展的方向发展。塑料机械工业所提供的装备主要是为信息、生物、建筑、包装、交通及家电等行业服务。 国外塑料机械工业的发展趋势大致包括以下几个方面。 (1)微型大型并重 :微型化是今后各类塑机产品的发展方向之一。微型塑机会有越来越多的市场需求，目前它在电子、信息、电器、医疗、生物等行业的初步应用，已经取得了阶段性成果，并呈良好的发展势头。目前，国际上已经成功地开发出了质量为万分之一克的注射制品成型加工设备、直径为1mm的塑料管生产设备和容积为3m l的中空吹塑机等微型塑机产品。据了解，一些发达国家目前正在研制能够替代人体血管的直径小于0．5mm的塑料管生产设备，而日本则表示要开发质量为十万分之一克的注射成型加工设备。 大型化也是塑机产品的发展方向之一。目前生产小轿车车身板用的8000kN注射机已经研制成功，直径为2000m m的塑料管、宽为10m的片材及容积为5000L的中空容器等大型塑料制品的生产设备也已投放市场。但是，一些工业用的更加大型的塑料制品的生产加工设备却依然是一片空白。如小型快艇、运动艇、洲际长途输液输气用的超大直径塑料管以及1万升甚至更大容积的塑料储装容器的生产所需的加工设备，都有待于研制开发。 (2)个性化 :长期以来，塑料机械的机型、功能、规格的固定不变，已经远远不能满足市场的需求。由于塑料制品需求的不断变化，塑料制品生产企业要求塑料机械工业能及时为其提供最具市场适应能力的塑料机械装备。如：动模板可移动又可转动的注射机、大成型面积小合模力或大合模力小注射量的注射机、混料与注射联用的成型设备、负有特殊外围配套装置的挤出生产线、木材与塑料共混成型加工设备等等，这就要求塑料机械工业企业及时对市场作出最有效的反应。 塑料机械工业产品的个性化发展，必然会淡化以往的成批量生产的模式，同时也会促进塑料机械工业产品的模块化设计和集成技术的应用，这就要求塑料机械企业在技术人才、技术创新方面要具有雄厚的实力，也要求企业能够在第一时间内，准确地把握客户的个性化需求。 (3)智能化 :塑料机械工业产品的自动化控制技术目前已经发展到相当高的水平，设备单元的自动控制、参数的闭环控制、过程联运在线反馈控制等电子与计算机技术都在塑料机械产品上得到了较为广泛的应用，目前已经出现了具有智能控制功能的塑料机械产品。人们有理由相信，随着智能化塑料机械产品的不断问世，为数众多的无人车间、无人工厂会变成现实。2002年1月30日

    近期国内塑料市场动态

    发表于 12月 31 2014

    高压聚乙烯：本周，国内高压聚乙烯市场在经历了近一年多的低迷之后，终于出现了一轮较为强烈的上涨，各生产企业普遍将出厂价格上涨200－300元/吨以上，而中间商和下游用户也纷纷入市吸货，造成整个国内市场的价格全面上扬，部分地区的涨幅已达300-500元/吨不等（最高市场价在6500元/吨左右）。虽然每年一月份都是高压市场的需求淡季，但今年的情况却正好相反，市场之火爆也出乎了很多业内人士的预料。但是，应该看到此次行情主要还是由于中间商人为炒作造成的，再加上今年中国正式加入WTO后，国家农用膜料进口配额被取消，致使部分原来依赖进口配额的中间商和下游加工企业转而寻求其它资源（尤其是国内资源），致使国内资源略显紧张，而且关税的下调导致进口产品一时无法进入中国市场，也使得市场出现了供不应求的局面。此外，前期国内出厂价格价格偏低，市场也有存在着一定的反弹要求。后市来看，由于线性高压与高压市场价格差距仅为200－300元/吨，因此国内高压市场仍有一定的上涨空间，但上涨趋势已不会太大，随着春节的日益临近，市场价格将会逐渐趋于稳定。　　燕化公司LDPE装置1线产注塑料1I2A，价格为5600元/吨，上涨200元/吨2线正在转产1F7B，价格未定，三线产1C7A价格为6100元/吨，并取消区域价。目前LDPE整体销售非常火爆，库存很低，有继续上涨价格的可能性。目前新高压再次开车，有可能在近期出合格品。 低压聚乙烯：本周，国内低压聚乙烯市场极为火爆，市场在各贸易商大力炒作之下，短短几天竟然上涨了有近600－700元/吨以上，部分地区的拉丝料市场的价格已涨之至5900－6000元/吨，低压市场一月份会出现如此大的行情，在其历史同期都可以说是史无前例的。此次行情可以说是由多方面因素造成的，一是由于年前各企业为完成任务，无一例外的清空了库存，导致各企业库存都很低；二是国内低压价格在经历了一年多的下跌之后，已到了跌无可跌的地位，市场有强烈的反弹愿望；三是中国加入WTO之后，进口产品纷纷上调其出口报价，再加上部分国外聚乙烯装置出现故障或者停产检修，都导致市场上进口资源量较少；四是由于前期市场低迷，国内各中间商的手中都不敢囤货，他们手中有大量闲散资金，因此一旦市场有所启动就纷纷入市吸货，导致市场出现供不应求的情况。后市来看，国内低压聚乙烯市场上涨的幅度已明显偏大，虽然还有可能进一步上涨，但风险已大，随时有上涨乏力的可能性出现。 燕化HDPE装置1条线产5000S，售价 5600元/吨，比上周上涨300元/吨，本周连续2次上调出厂价格，目前销售仍非常顺畅，库存很少。另1条线产中空料5200B，价格为5750元/吨，比上周上调450元/吨，销售也较好，库存不多，而其它牌号均无货。 聚丙烯：元旦过后国内市场出现明显反弹，尤其是拉丝市场供需两旺，厂家库存急剧减少，大量现货被中间商“哄抢”，以应春节后需求恢复、价格有望反弹之需。与此对应，共聚料、膜料、纤维料市场及企业价格均没有过旺反应。韩国、台塑、福聚等PP厂商因中国市场强劲的需求而受益，大量1月现货已经订购一空，2月报盘全面上扬。1月底-春节前，国内塑料市场热销局面将可能逐渐降温。 　　燕化PP装置1线生产拉丝料2401，售价4900元/吨（有区域价），比前期上调200元/吨，2线生产洗衣机专用料1947，价格5500元/吨（有区域价），3线产薄膜料F1002，价格为4700元/吨。目前，拉丝料S1003价格也为4900元/吨（有区域价），上涨200元/吨，其它牌号价格暂时无变化。目前各品种中拉丝料销售情况较好，下周有可能继续上调价格，其它品种销售也较好，总体库存都不高。 聚苯乙烯：本周，国内聚苯乙烯市场继续好转，在各企业上调出厂价格的基础上，各地市场的价格也有进一步的上涨，涨幅普遍在100－300元/吨左右。目前市场成交比前期变得更加红火，许多贸易商和最终用户都已开始加大采购力度，购销两旺的局面已然形成。近期市场之所以表现的较好，主要是由于国内部分聚苯乙烯生产企业因故停车，导致市场供应有所减少，再加上苯乙烯市场仍处于较高水平，也促使聚苯乙烯市场有所反弹，此外，目前进口报价在500美元/吨左右，折合人民币比国内市场价格高200－300元/吨左右，这也是对市场有一定的刺激作用。后市来看，国内聚苯乙烯市场虽然还有一定的上升空间，但受苯乙烯市场有所走软的影响，市场继续大幅上涨的可能性微乎其微，在高位横盘整理的可能性较大。 燕化PS因装置故障暂时停车，透苯666D出厂价格为5200元/吨，无库存而前期转产的688C(双向拉伸板材料)，售价为5400元/吨，销售一般。 来源:全国塑料加工工业信息中心 发布日期:2002-1-28 2002年1月28日

    纳米塑料研制成功

    发表于 12月 31 2014

    由中国科学院化学研究所研制出一种新型纳米粘土母料，并对纳米粘土母粒与尼龙熔融共混制成性能优良的PA6/n-MMT纳米塑料。 中国科学院化学研究所开发成功纳米复合PET工程塑料。 美国GF公司首先用纳米PET塑料，生产啤酒瓶。这种耐冲击、高耐压、抗菌的纳米塑料啤酒瓶在市场上受到普遍欢迎。 清华大学高分子研究所，研制成用纳米SiO2改性高密度PE获得了比普通高密度PE耐磨性大大提高的纳米塑料。 2001年12月14日

    国内领先BOPP激光防伪收缩膜

    发表于 12月 31 2014

    由海南现代企业股份有限公司研发成功的BOPP激光全息防伪收缩膜近日通过国家科学技术成果鉴定。该产品与国内其他防伪材料和防伪技术相比，具有不易仿造，易于普通消费者识别、环保无毒、不影响被包装产品本身性能与功用、外形美观等特点。该产品成功解决了BOPP收缩膜生产配方及压印工艺等技术难题，为国内首创，工艺技术居国内领先水平。 BOPP激光全息防伪收缩膜是含多项专有技术的新型防伪产品。该产品通过在包装薄膜表面直接制取防伪标志图案，清晰直观，极易于消费者识别，有利于从根本上堵死假源，打击制售假货的不法行为，维护品牌形象和厂家的根本利益，是国内防伪技术的重大突破；而且该防伪产品还能在不改变包装防潮、保香、外观设计等性能的情况下增强产品包装的外观美感。BOPP激光全息防伪收缩膜可应用于卷烟、医药食品、及音像制品等行业的产品防伪包装。2002年1月25日

    温州民企富有魅力、充满挑战　上市公司老总跑来“打工”

    发表于 12月 31 2014

    　 放着上市公司的总裁职务不干，却跑到温州民营企业“打工”———上演这一幕的两位“主角”昨天首次出现在天正集团数千名职工面前。 　　这两位“主角”是几个月前还是国嘉光电股份有限公司总裁的何捷和国电南京自动化股份有限公司总裁的程传玲。何捷最擅长的是资本运营和企业发展战略的制定；而程传玲则是科技开发和管理皆精的专家，由他经营多年的“国电南自”被评为上市公司50强。 　　何捷、程传玲与“天正”结缘始于中欧国际工商管理学院，在这个亚太地区最好的工商管理学院，他们与“天正”董事长高天乐是同班同学。程传玲说，他们相互欣赏、敬佩对方的才学和为人，同时，他和何捷发现温州民营企业的机制和活力，是许多上市公司也难以做到的，便萌发到“天正”干一番事业的念头。 　　而曾被外资企业邀请的何捷则称，他们更喜欢接受挑战，现在，外资企业各方面都很成熟，而温州的民营企业正在深化改革，有许多事情可做。 　　何捷、程传玲现在“天正”分别担任执行总裁、副总裁。他们开始在推进“天正”上市和技术进步方面发挥了重要作用。 　　以前曾忙得团团转的高天乐现在显得很轻松，他说，上市公司总裁的水平确非一般，一些思路让人深受启发，更重要的是他们进一步规范了企业的发展，以前一些民营企业制定发展战略时，常信口开河：明年销售额达50亿元。也不管是否能做到，反正也没人管。现在，他们制定了切实的目标、计划和实施步骤。 　　在温州，外地上市公司的总裁辞职前来尚属首例。有消息说，何捷、程传玲在“天正”的年薪可能称不上新闻，但拥有比较诱人的期权。 　　●王若江 2002年1月24日
