    塑料原料市场关注关税下调

    发表于
    12月 31 2014

    2002年1月1日起我国将降低关税。塑料原料关税下降基本幅度在2个百分点以内，其中PP的关税下调幅度最大,将从16％左右下调到10％。而有些塑料制品则也达到4个百分点，详见关税减让八、关税减让九、关税减让十。到2008年，聚乙烯、聚丙烯、聚苯乙烯、聚氯乙烯和ABS的关税将从目前的16％左右降至6.5％。对现在的低迷塑料原料市场将不是一个好消息，国内部分产品成本已接近进口产品的到岸价，塑料原料行业产品明显缺乏竞争力，在逐步取消进口配额的情况下，这些产品的价格会进一步受到国际市场的影响。 关税的下调给供应商和进口商提高价格留出了一定的空间.由于1月份的塑料产品成本核算方法被打破，新的方法仍未出台。目前，大部分国外商社仍在等候总部对报盘的定位，而且近日国内石化厂家迫于销售压力，纷纷下调出厂价，部分商社因此暂停对外报盘。目前LG商社12月份的产品已销售一空，1月份的报盘将在12月中旬公布。日本PVC出口商在12月10日正式对外宣布新一年的报价。新的报价里美国信越PVC报价为420美元/T 。 CIF中国主港。日本信越TK系列PVC树脂的报价为440美元/T；比12月份报价有所上扬。生产和流通环节都曾期望关税的下调将刺激进口商们对较低的库存重新补充，但这一点表现得尚不明显。 　　关税在下调，似乎国外塑料原料进口成本将下降，但也有“意外”，曾经以低价格冲击我国北方市场的俄罗斯PVC进口成本增加300元/T左右，这是由于中国加入世贸后将逐步调整、废止部分边贸政策以适应世贸规则的要求。据有关部分宣布从2002年第一季度开始对俄边贸将不能享受增值税半税先征后退的优势。 2001年12月25日

    塑料食品袋有话说

    发表于
    12月 31 2014

    　　全球塑料食品袋的使用量与日俱增，而对于用塑料制品包装食品有无毒的急诊却愈演愈烈。塑料食品袋到底给我们的生活带来的是便利还是祸害？应该给它一个公正的回答，因为至少在短时间内我们生活已经离不开它了。　　媒体：主张摒弃塑料食品袋　　近日，中央电视台报道了某些医疗单位把废旧一次性医疗用品卖给个体塑料加工户，由他们生产出带有病菌、危害人体健康的塑料袋。于是，把已成媒体众矢之的塑料食品袋再次推向被告席。　　许多媒体纷纷发表言论：塑料本身人释放有害气体，容易导致食品（尤其是熟食）变质，人吃了后会引起呕吐、腹泻等症状；现在市场上使用的大部分是聚氯乙烯塑料食品袋，燃烧后残存空气里的氯化物及重金属离子会严重地危害人类健康，破坏生态环境；绝大多数由小型企业或地下加工厂生产的再生料食品袋，未经消毒处理，便大量地流入市场，成为包装的主导用品，而这些再生料含有严重超标的病菌和致癌物。有关人士反复呼吁，购买食品最好使用纸、布袋盛放。　　专家：还塑料食品袋清白　　专家认为，媒体报道中许多模糊的概念，给方便人们生活，提高人们生活质量和满足食品包装等方面做出较大贡献的塑料食品袋加上莫须有的罪名，给塑料行业的发展前景蒙上一层浓厚的阴影。这些错误报道给不明真相的消费者带来疑惑，也给塑料食品袋生产企业带来不必要的经济损失。　　事实上，塑料食品袋是一种安全实用的包装。我国从20世纪80年代起禁止使用含有增塑剂的软质聚氯乙烯塑料制品食品包装袋，而目前市场上的食品包装袋主要材质是聚乙烯、聚丙烯、聚酯和尼龙。这些塑料的分子结构中根本没有氯化物和重金属离子，因此，在燃烧时不会放出危害人类健康的气体。　　另外，食品变质中毒的原因的多方面的，没有选用适宜的包装材料和包装技术虽然是主因，但没有根据食品包装法的要求注明保质期或消费者不按标注的保质期而过期使用也是原因之一。因此，“中毒”纯粹是塑料食品袋使用不当造成的，而非塑料包装材料本身的责任。当前国内市场用的塑料食品袋绝大部分是由大型国有、集体、民营和三资企业生产的，当然，一些农贸市场及小店铺里也混杂了一些非法地下加工厂生产的用回收料加工的国家明令禁止的食品包装袋，但这不应该和合格的塑料食品袋混为一谈，不能因噎废食，而应加强市场监督。　　至于环保问题，应通过加强回收力度和开发可降解塑料相结合的办法来解决。无论塑料食品袋、纸袋、布袋均各具特性，其用途也应各有自己的定位，应根据包装内容物的要求各得其所。　　消费者：我们图方便，但要安全　　农贸市场，超市里的塑料食品袋用量大得惊人，而消费者是否考虑到有毒无毒的问题呢？“没有了塑料袋，我们怎么拿东西”？这是记者在家乐福超市采访时，一位姓张的大妈说的。天福超市的吴小姐告诉记者：“我们图方便，如购买带酱汁类食品，还是塑料袋好用。但是安全因素还是要考虑的。”　　如今，塑料袋代替了以前的菜篮子，完全拒绝使用似乎不现实，也不见得合乎国情。但如何规范生产，合理使用，是一个值得重视、思考的问题。这不仅是对消费者负责，也是对塑料生产企业和整个行业负责。2001年12月21日

    中国实现合成树脂进口货源多元化

    发表于
    12月 31 2014

    过去中国进口合成树脂对日本和韩国产品的依赖程度高，如20世纪90年代中期前，自韩国的进口量占进口总量的50%。中国努力实现合成树脂进口货源多元化，自1998年韩国产品下降为占30.6%，今年下降为占23.4%，而聚乙烯、聚丙烯韩国产品下降为占14%。与此同时，我自台湾省、东南亚、中东国家的进口量逐年大幅增长，它们的产品在中国进口总量中上升为占33%以上。 2001年12月20日

    国家确定优先发展141个高技术领域

    台湾塑料原料１２月份价格全面下跌

    发表于
    12月 31 2014

    由于台湾中油公司调降乙烯和丙烯１１月份的结算价格，台湾地区１２月份塑料原料的销售价格全数下跌。台湾氯乙烯厂商买的乙烯价格为每吨３８９美元，较上月下跌５１美元；聚乙烯厂商买到的乙烯价格为每吨３６６美元，较上月下跌５０美元；聚丙烯厂商买到的丙烯价格为每吨３２７美元，较上月下跌３６美元。　　在此情况下，包括台塑公司和奇美实业在内的台湾塑化原料厂商１２月初普遍调低了岛内价格，其中PVC为每公斤新台币(约３４．５新台币合１美元)１６.２元，下跌１元；HDPE为１８.５－１９.５元，下跌２.２元；LDPE为２０.２－２３.３元，下跌２.１元；PP为２１元，下跌０.９－１.４元；PS和ABS树脂每公斤也分别下跌了１.５－２元。值得注意的是，岛内苯乙烯单体１１月份的结算价格为每吨３９４.４美元，大幅下跌了５３.６美元，不但创了今年以来的新低，也是自去年３月以来的最低价格。　　台塑公司认为，１２月份岛内塑化原料及上游原料价格下跌的主因除美国“９·;１１”事件的后续影响外，大陆的买盘不振也是重要因素。近日市场传闻大陆可能将关税由１６％降至１２％，而且还将取消进口批文的审核，使大陆厂商暂不进料，造成了塑料原料价格下跌，估计包括PE和PVC在内的远东塑料原料厂商已减产一至二成。 2001年12月19日

    国际包装市场的新品开发

    发表于
    12月 31 2014

    　　随着包装科技的飞速发展，近年来国外又相继开发出一批功能独特、品种多样的包装新品，令人耳目一新，也受到了包装业界的关注。 　　生态包装　法国达诺内奶制品公司近来向公众展出了一种从甜菜中提取乳酸制成的酸奶盒和杯子。这种生态盒（杯）的基本原料是甜菜，通过发酵转化为乳酸，经抽出水分形成丙酯，再加工为聚乳酯，掺入矿物质后便成为一种轻型坚固的材料。用这种材料制成的包装盒（杯）可在55℃下，60天内分解为农家肥料。　　微波包装　开发微波包装是欧美国家食品包装的新趋势。德国MatonPack公司、瑞典Sprinter公司生产出了扁盘式微波包装。该包装选用C－PET材料，采用PET衬里，多层复合，并压制出加强筋。 　　采用这种包装的食品从冰箱取出就可直接进入微波灶烘烤，可耐受230℃高温。由于欧美国家使用微波炉的家庭已达90％，微波包装发展很快。Pillsbury公司还研制出一种烤饼包装，为扁盘形状，印刷装潢精美，深受欢迎。 　　抗菌包装　日本一家化学工业公司研制成功一种抗菌塑料包装容器。该包装容器在食品、化妆品生产和制药业等领域有着广泛的用途。这种抗菌塑料包装品，是用纤维化塑料和聚丙烯等合成材料与一种用于食品薄膜的抗菌剂混合制成的，能防止微生物和细菌的繁殖，防止所盛装的食品、化妆品或药物变质，大大延长了保质期。 　　防水包装　澳大利亚研制出为鱼肉、水果和蔬菜等含水农产品专用的包装纸盒，这种纸盒内外均覆盖或涂敷聚丙烯或聚乙烯，能防水，在包装时无需用其它防水材料。该包装纸盒适用于需要冷冻或冷藏的农产品，可使它们在销售过程中始终保持一定水分，防止因冻结或潮湿而受损。此外，该纸盒坚固耐用，经得起长途运输和多次装卸，在上面印上各种图案和商标后，还可用于出口物品的包装。 　　耐热包装　日本一家塑料公司最近研制成功了一种用于包装快餐食品的新型耐热塑料包装材料。该包装材料除了加工性能好，没有毒性以外，最主要的特点是耐热，其变形温度高达280℃以上。用这种材料盛装方便面，冲入沸水后，包装盒不会变形，并能完全保持快餐食品的风味。 　　热缩包装　国外流行的这种热缩包装，是利用一种具有热缩性能的塑料薄膜作为包装材料，将物品包好后，迅速放到一个加热装置中加热到一定温度，包装薄膜即自行按一定比例收缩，紧紧裹住被包装物。热缩薄膜一般很薄，透明度好，可以直接观察被包装物。热缩包装用途广，可用于食品、工业品及其他商品，也可用于包装多件商品。该包装具有运输方便，透明度好，外观好看，包装牢固，减少包装体积，保持清洁等优点。 2001年12月19日

    大庆市５０００吨／年抗菌塑料项目招商

    发表于
    12月 31 2014

    该项目的主要产品为抗菌ＡＢＳ树脂。近年来塑料制品在人们的生活和生产活动中已经得到越来越广泛的应用。塑料制品几乎是遍布我们生活的每一个角落。塑料产品在给人类带来方便的同时也成为有害细菌传播的最便捷的“交通工具”。抗菌塑料就是将少量的抗菌剂添加到塑料中，使塑料具有抗菌能力，这种塑料能够抑制和杀灭有害细菌。适用范围很广，目前多用于家用电器、通信器材、汽车制造等方面。 市场分析：随着人们环保意识的提高，人们越来越重视对自己生存环境的保护。在欧美一些发达国家，人们早已在电话的听筒、电脑的键盘、公交车的扶手上或多或少地使用了抗菌材料，有害细菌在这些地方再也找不到生息繁衍之地。 目前抗菌塑料已经开始用于大型家电（如冰箱等）。具有抑制和杀灭有害细菌的家电产品（如抗菌冰箱）的开发和生产将带动中国的家电业从传统家电型向健康家电型的换代升级。抗菌塑料是一个崭新的市场，一个亟待开发的市场，同时也是一个市场前景十分广阔的新领域。据不完全统计，目前仅冰箱这一领域每年就需要抗菌塑料１０多万吨。 2001年12月18日

    本报讯 在温州商人姚学义的组织下，一座由温州人投资、占地600亩、总投资2亿元、东南亚最大的商品批发集市将在越南首都河内市动工兴建。姚学义昨天向记者透露了这一消息。 　　姚学义1997年离开温州到四川经商，成立了一家市场开发管理公司，现为成都荷花池商会会长。今年初，商会在讨论今后发展计划时分析，越南的大小商铺中陈列的商品，70％是中国的中低档商品，但价格却比国内高出50％以上。姚学义等人便决定在越南首都河内市建一座东南亚最大的综合型批发市场———中国商品批发城。该市场可通过越南辐射东南亚的缅甸、老挝、柬埔寨等国家。 　　姚学义说，“中国商品批发城”的启动资金需2000万元，内设服装、皮鞋等30个左右的专业市场，主要销售中国的中低档商品。目前，他们已与河内市政府初步达成共识，2000万元启动资金也已筹齐。 　　2001年12月18日
